# Women's Day Celebration Website

A beautiful single-page website created to celebrate International Women's Day (March 8th).

## Features

- Full-screen background image
- Background music that plays when the page is interacted with
- Beautiful animated greeting text
- Interactive question with fun button behavior
- Responsive design that works on all devices

## How to Use

1. **Setup:**
   - Place your background image in the same folder and name it `background.jpg`
   - Add your music file in the same folder and name it `music.mp3`

2. **Viewing the Website:**
   - Simply open the `index.html` file in any modern web browser
   - Click anywhere on the page to start the music (browsers require user interaction to play audio)
   - Enjoy the animations and interactive elements

3. **Customization:**
   - You can modify the text in the `index.html` file
   - Change colors and styles in the `style.css` file
   - Adjust animations and interactions in the `script.js` file

## Browser Compatibility

This website works best in modern browsers such as:
- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari

## Note

Due to browser security policies, audio will only play after the user interacts with the page (clicks anywhere on it).
